# Teste para Vaga Front-End AngularJS

## Requisitos

* Ter o Node na máquina
* Ter o Gulp na máquina. Caso não tenha instalado, basta rodar o seguinte comando "npm install --global gulp-cli"

## Setup

Será necessário rodar o seguinte comando "npm install" dentro do projeto via terminal. Feito isso, rodar o comando "gulp", que irá gerar os arquivos da aplicação.

Agora é só acessar o projeto no browser.
